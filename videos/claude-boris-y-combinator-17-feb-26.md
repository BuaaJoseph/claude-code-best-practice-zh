## 演讲内容

[`0:00`](https://youtu.be/8YRj2m6WEtc?t=0) 大家好，我是 Boris。我是 Claude Code 的创始人，也是 Anthropic 的工程师。在加入 Anthropic 之前，我在 Meta 工作了 7 年，负责代码质量方面的工作。现在我在 Anthropic 工作，主要负责 Claude Code 的开发。Claude Code 是一个 AI 编程助手，它可以帮助你编写、审查和调试代码。

[`0:30`](https://youtu.be/8YRj2m6WEtc?t=30) 今天我想和大家分享一些关于 Claude Code 的故事，以及我们如何构建这个产品。我们从一个简单的想法开始：我们想要构建一个能够帮助工程师更高效地工作的工具。我们注意到，许多工程师在编写代码时，会花费大量时间在重复性的任务上，比如查找代码、修复 bug、编写测试等。我们想要自动化这些任务，让工程师能够专注于更有价值的工作。

[`1:00`](https://youtu.be/8YRj2m6WEtc?t=60) 我们首先构建了一个原型，它可以在终端中运行，并且可以与代码库进行交互。我们发现，如果给模型提供正确的工具和上下文，它可以非常有效地帮助工程师完成各种任务。我们还发现，模型非常喜欢使用工具，只要给它一个工具，它就会想出如何利用这个工具来完成任务。

[`1:30`](https://youtu.be/8YRj2m6WEtc?t=90) 例如，我们给模型提供了 bash 工具，它就可以自动使用 bash 命令来执行各种操作。我们还提供了文件编辑工具，模型可以使用它来读取和修改文件。我们发现，只要给模型足够的上下文和工具，它就可以完成非常复杂的任务。

[`2:00`](https://youtu.be/8YRj2m6WEtc?t=120) 我们还发现，模型可以非常好地处理代码审查。我们有一个内部工具，可以在提交代码之前自动进行代码审查。它可以检查代码中的问题，并提供修复建议。这大大提高了我们的代码质量。

[`2:30`](https://youtu.be/8YRj2m6WEtc?t=150) 现在，Claude Code 已经被广泛使用。我们估计，在 Anthropic，Claude Code 写了大约 80% 的代码。我个人每天会提交 20-30 个 PR，而 Claude Code 写了所有这些代码的 100%。我没有手动编辑过任何一行代码。这真是太神奇了。

[`3:00`](https://youtu.be/8YRj2m6WEtc?t=180) 我认为，我们正在经历一个类似于 15 世纪印刷术的变革时期。在那个时期，抄写员是一种专门的职业，他们负责手写书籍。但是，随着印刷术的发明，抄写员的职业发生了变化，他们成为了作家和作者。市场的扩大导致了新的职业的出现。

[`3:30`](https://youtu.be/8YRj2m6WEtc?t=210) 现在，我们正在经历类似的变化。软件工程师曾经是一种专门的职业，他们负责编写代码。但是，随着 AI 的发展，任何人都可以编写代码。这并不意味着软件工程师会消失，而是意味着他们的角色会发生变化。他们将成为更有价值的角色，比如产品经理、设计师、或者创业者。

[`4:00`](https://youtu.be/8YRj2m6WEtc?t=240) 我们还发现，Claude Code 正在被用于各种非编程任务。例如，我们的财务团队使用它来进行预测，数据科学家使用它来进行分析，销售团队使用它来进行客户管理。这说明，Claude Code 不仅仅是一个编程工具，而是一个通用的 AI 助手。

[`4:30`](https://youtu.be/8YRj2m6WEtc?t=270) 我们还推出了一个新的产品，叫做 Co-work。它是专门为非技术人员设计的。它可以在虚拟机的环境中运行，这样可以确保安全性。用户可以通过自然语言与它交互，它可以帮助他们完成各种任务，比如管理文件、发送邮件、安排会议等。

[`5:00`](https://youtu.be/8YRj2m6WEtc?t=300) 我们相信，未来会有更多的形式因素出现。我们正在尝试不同的方式来让用户与 Claude Code 交互，比如网页版、移动端、IDE 插件等。我们相信，随着模型的不断改进，我们能够为用户提供更好的体验。

[`5:30`](https://youtu.be/8YRj2m6WEtc?t=330) 最后，我想说的是，我们正在经历一个非常激动人心的时期。AI 正在改变我们工作和生活的方式。我们很高兴能够成为这个变革的一部分。我们将继续努力，构建更好的产品，帮助人们更高效地工作。

[`6:00`](https://youtu.be/8YRj2m6WEtc?t=360) 谢谢大家！
